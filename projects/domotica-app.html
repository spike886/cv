Mobile App para domotica

  • Marzo 2014
  • JavaScript Cordova AngularJS Ionic Bootstrap 3 WebSocket REST Facebook Bluetooth
  • Información no divulgable

Aplicación móvil para Android, iOS y Windows Phone, desarrollada con Cordova.

Permite el control, uso y mantenimiento de equipos de domotica. Especialmente diseñada para el manejo de equipos de tipo spa.

Capacidad de sincronización con servidor. Conexión live con equipos a traves de websocket y Bluetooth. Autenticación de usuario a traves de Facebook
